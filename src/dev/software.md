# 소프트웨어

## 실제 카드단말기 연결
 양산으로 들어가는 펌웨어는 KICC사의 ED-785 단말기를 기준으로 하고 있습니다. 하지만 NDA 문서에 의거하여 해당 코드는 공개된 코드에 없으며 dependency injection 을 통해 따로 관리되고 있습니다. 따라서 공개된 소스코드에는 실제로 동작하지않는 예제 프로토콜 코드만 담고있습니다.
